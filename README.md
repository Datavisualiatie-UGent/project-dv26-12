# Project Datavisualisatie: How does AI shape the current development landscape
*based on Stack Overflow Annual Survey 2025*

## What to visualize

### Exploration: Bar chart exploring all questions
Plot each question result for a chosen category representation (age, dominent programming language, education, salary bracket, etc)

### AI favorability: vertical category percentage bar chart
*How favorable is your stance on using AI tools as part of
your development workflow?*
plot bars representing the percentage of answer of different categories (colorring green to red)
Goal: introduction to the AI landscape for specific groups developers. 

### AI specific model usage: Correlation matrix
*Which LLM models for AI tools have you used for
development work in the past year, and which would you
like to use next year? Select all that apply.*
Correlation plot (axis is all models used + all models you want to use)
Goal: visualize pattern between usage of certain models and the desire to use other specific models.

### AI integration: Radar chart 
*Which parts of your development workflow are you
currently integrating into AI or using AI tools to accomplish
or plan to use AI to accomplish over the next 3 - 5 years?
Please select one for each scenario.*
plot radar chart showing the current (or incentive to) integration of AI.
Goal: show current integration in aspect of development.
